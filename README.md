# 🌤️ Weather App

A simple and responsive Weather App built using HTML, CSS, and JavaScript.

## 🚀 Features

* Search weather by city name
* Live weather data using Open-Meteo API
* Temperature display
* Wind speed display
* Weather icons
* Error handling
* Enter key support
* Responsive UI

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API
* Open-Meteo API

## ⚙️ How It Works

1. User enters city name
2. Geocoding API fetches latitude and longitude
3. Weather API fetches current weather data
4. Weather details display dynamically on the screen

## 📸 Output

* Weather condition
* Temperature
* Wind speed
* Latitude & Longitude

## 🌐 API Used

* Open-Meteo Geocoding API
* Open-Meteo Weather API
